
# wordlink
- long link -> sequence of words.
- sequence of words -> long link.

- [rust and cargo](https://www.rust-lang.org/tools/install)

- [docker](https://docs.docker.com/get-docker/)
- [docker-compose](https://docs.docker.com/compose/install/)
